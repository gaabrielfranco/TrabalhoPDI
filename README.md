# Software de Processamento Digital de Imagens

Esse é o software resultante do trabalho da disciplina INF 394 - Processamento Digital de Imagens, da UFV. Todo software foi implementado com base em um esqueleto disponibilizado pelo professor Marcos Henrique. O trabalho ainda está em evolução.

# Pre-Requisitos
Esse software é implementado em Python3 utilizando o pacote Anaconda. Segue os links de instalação:
  - [Download do Python3][ps]
  - [Download do Anaconda][as]

### Execução

```sh
$ python gui.py
```
